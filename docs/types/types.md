# types.ts — Shared interfaces and constants

## What it does

This file defines the TypeScript interfaces and constants shared across the
project. It is the typed contract that keeps parser, navigation, rendering,
hitmap, and UI state aligned.

## Why it exists

Keeping shared types in one file:

- avoids logic-level circular dependencies
- provides one reference point for the data model
- makes it possible to import only the types a module needs

## Interfaces

### BBox

```ts
export interface BBox {
  x: number;
  y: number;
  width: number;
  height: number;
}
```

Bounding box of a region in SVG coordinate space. Used by camera targeting,
dynamic tiles, and geometry-related UI state.

### Region

```ts
export interface Region {
  id: string;
  bbox: BBox;
  level: number;
  alias: string;
  parentId: string | null;
  strokePadding: number;
}
```

Described in detail in [Region.md](../parser/Region.md).

### NavState

```ts
export interface NavState {
  level: number;
  focusedId: string | null;
  history: NavHistoryEntry[];
  skippedLevels: string[];
}
```

`skippedLevels` stores the singleton levels crossed automatically by the most
recent resolved drill-down.

Described in detail in [NavState.md](NavState.md).

### NavHistoryEntry

```ts
export interface NavHistoryEntry {
  id: string | null;
  level: number;
  camera: { scale: number; translateX: number; translateY: number };
}
```

Snapshot of a previously visited visual state, stored in `NavState.history`.

### RasterCache

```ts
export interface RasterCache {
  low: HTMLCanvasElement | null;
  mid: HTMLCanvasElement | null;
}
```

Described in detail in [RasterCache.md](RasterCache.md).

### DynamicTile

```ts
export interface DynamicTile {
  canvas: HTMLCanvasElement;
  bbox: BBox;
  scale: number;
  padding: number;
}
```

On-demand raster cache for a focused region, generated by `buildDynamicTile()`
and consumed by the renderer.

### BreadcrumbItem

```ts
export interface BreadcrumbItem {
  id: string;
  label: string;
  level: number;
  isSkipped: boolean;
}
```

Represents one breadcrumb node built from the current focused path.

- `id` is the real region ID
- `label` is the humanized breadcrumb label
- `level` is the hierarchy depth
- `isSkipped` marks levels preserved conceptually but crossed automatically by
  singleton skip

## Constants

```ts
export const LEVEL_NAMES = ['ps', 'lf', 'ph', 'ci'] as const;
export const LERP_FACTOR = 0.08;
export const SNAP_THRESHOLD = 0.5;
export const MAX_CANVAS_DIM = 8192;
export const BG_COLOR = '#0a0a0a';
export const DIM_ALPHA = 0.15;
```

| Constant | Value | Used by | Description |
|---|---|---|---|
| `LEVEL_NAMES` | `['ps', 'lf', 'ph', 'ci']` | `hud.ts` | Human-readable level aliases for debug output |
| `LERP_FACTOR` | `0.08` | `camera.ts` | Fraction interpolated per animation frame |
| `SNAP_THRESHOLD` | `0.5` | `camera.ts` | Threshold used for the final snap |
| `MAX_CANVAS_DIM` | `8192` | `rasterizer.ts` | Safe upper bound for canvas dimensions |
| `BG_COLOR` | `#0a0a0a` | `renderer.ts` | Canvas background color |
| `DIM_ALPHA` | `0.15` | `renderer.ts` | Dimming opacity in focused mode |
